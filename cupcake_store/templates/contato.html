Sobre o Projeto

Este projeto foi desenvolvido por Guilherme Borba, como requisito para a conclusão da disciplina de PIT II do curso de Engenharia de Software da Faculdade Cruzeiro do Sul.

Este site tem como propósito apresentar uma vitrine virtual de cupcakes, permitindo que você navegue por diferentes categorias, visualize imagens dos produtos, leia suas descrições e adicione itens ao carrinho de forma prática e interativa.

Este projeto foi desenvolvido com:

  • Python (Flask) para o back-end
  • HTML para a estrutura das páginas
  • CSS para o design e estilo das páginas

O desenvolvimento foi realizado na IDE PyCharm, com controle de versão pelo GitHub.
